## 題目:  
>今天有一個脾氣不好的教授  
如果準時抵達的人數低於教授的期望值   
這節課就會被取消  

### Input
- ary 是一個`無序`陣列 長度為 6~30之間 
- k 是一個正整數為 (2~ ary.length/2  之間)

陣列裡面會塞對應長度數量的數字   
每個數字為(-25 ~25之間)  
如果數字為負代表他提前抵達  
如果數字為0代表他準時抵達  
只要`提前`或是`準時`都算是準時抵達人數  
k為教授的期望準時的人數

### Output
- 如果課程`被取消` 輸出 'YES'
- 如果課程`沒被取消` 輸出 'NO'

### 範例
```
ary =  [1,-1,-15,-16,4,-25,27]
k = 3
```
>小於0的有4個  
代表準時或是提早的人有4個  
4大於k  
所以輸出 "NO"

- if/else
